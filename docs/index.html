简介

An Introduction to R (中文GitBook)

R导论是An Introduction to R的中文版。由Guohui Ding (丁国徽)，在2004～2006年翻译完成。2017年他应网友要求重新制作了PDF版，并将底稿开源。

原译者希望将有兴趣者一起将新版改写成Markdown格式，以GitBook的方式发布。
